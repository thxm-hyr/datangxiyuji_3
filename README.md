# 《大唐西域记》玄奘西行旅程网站

这是一个基于玄奘法师《大唐西域记》的交互式网站，通过现代技术重现玄奘法师的西行路线和见闻。

## 网站功能

### 🗺️ 交互式地图
- **可视化展示**：38个国家的精确地理位置标记
- **路线追踪**：玄奘法师西行路线的动态展示
- **分类显示**：按重要城市、佛教中心、普通国家分类标记
- **点击交互**：点击标记查看国家基本信息，跳转详情页

### 📖 国家详情页面
- **完整信息**：每个国家的历史沿革、地理位置、文化特色
- **原文对照**：《大唐西域记》原文与现代汉语翻译对照
- **图片展示**：相关遗址和文物的高清图片
- **佛教概念**：点击高亮的佛教术语查看详细解释

### 🏛️ 专题页面
- **昭武九姓**：详细介绍中亚粟特人建立的九个城邦国家
- **宗教文化**：西域地区的多元宗教发展与融合

### 🎨 古风设计
- **传统风格**：仿古卷轴、木纹背景的中国风设计
- **优雅字体**：使用宋体等传统中文字体
- **动画效果**：流畅的交互动画和页面过渡
- **响应式布局**：支持电脑、平板、手机等多种设备

## 技术特色

### 前端技术
- **HTML5/CSS3**：语义化标签和现代CSS特性
- **JavaScript ES6+**：模块化编程和现代语法
- **Leaflet.js**：开源地图库实现交互式地图
- **响应式设计**：移动端优先的适配方案

### 交互功能
- **地图标记**：自定义图标和弹窗信息
- **路线动画**：玄奘西行路线的动态绘制
- **佛教概念卡片**：点击术语显示解释的交互设计
- **页面过渡**：流畅的页面切换动画

## 使用说明

### 快速开始
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```

### 主要页面导航
- **首页**：项目介绍和统计数据
- **西行路线图**：交互式地图，点击国家标记查看详情
- **昭武九姓**：中亚粟特城邦专题介绍
- **宗教文化**：西域多元宗教发展历史

### 地图操作
- **显示所有国家**：查看所有38个国家的标记
- **显示行进路线**：只显示玄奘西行路线
- **重置地图**：恢复默认视图
- **点击标记**：查看国家基本信息，跳转详情页

## 数据来源

- **地理位置**：基于《大唐西域记》和各国家信息文件提取
- **原文内容**：来自"原文分段"文件夹的文言文和翻译
- **图片资源**：项目中的"图片"文件夹
- **佛教概念**：基于"buddhist_concept.csv"文件

## 开发说明

### 自定义数据
要添加新的国家或修改现有数据：
1. 在 `js/map-data.js` 中更新国家坐标和信息
2. 在 `js/country-data.js` 中完善国家详情
3. 添加对应的图片到"图片"文件夹

### 样式定制
主要样式定义在 `css/style.css` 中：
- 颜色主题：修改 `:root` 中的CSS变量
- 字体设置：调整字体堆栈
- 布局响应：修改媒体查询断点
